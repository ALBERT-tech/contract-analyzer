# Анализатор договоров — локально в браузере

Однофайловое веб-приложение, которое анализирует договоры и госконтракты **полностью в браузере** — без отправки данных на сервер. Модель (WebLLM на WebGPU, семейство Qwen2.5 / Qwen3 Instruct) скачивается один раз и кэшируется, дальше работает офлайн.

**Демо:** https://ALBERT-tech.github.io/contract-analyzer/

## Как работает

1. Вставляете текст или прикрепляете `.docx` (парсинг через mammoth.js).
2. Документ режется на пункты.
3. Под вопрос отбираются только релевантные пункты по ключевым словам (скоринг по корням слов, подтяжка заголовков разделов) в пределах бюджета (~3000 знаков).
4. В модель уходят только эти фрагменты + системный промпт — так обходится окно WebLLM в 4096 токенов.

Готовые вопросы: срок поставки, место поставки, обеспечение исполнения контракта, обеспечение гарантийных обязательств, сумма контракта. Плюс поле для своего вопроса (ключевые слова к нему подбирает сама модель).

Системный промпт требует отвечать только по переданным фрагментам, формат «ответ → Цитата: пункт», иначе «В договоре не указано».

## Требования

Chrome / Edge / Brave (desktop) с поддержкой **WebGPU**. Первая загрузка модели — 2–6 ГБ, дальше из кэша браузера.

## Стек

Чистый HTML / CSS / JS, [`@mlc-ai/web-llm`](https://github.com/mlc-ai/web-llm), [mammoth.js](https://github.com/mwilliamson/mammoth.js). Сборки нет — открывается как обычная веб-страница.
